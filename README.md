# Skill Chart

## 프로젝트 한 줄 요약

CSS 애니메이션을 활용해 퍼센트 기반 스킬 차트를 구현한 UI 프로젝트로,  
페이지 진입 시 프로그레스 바가 순차적으로 채워지는 애니메이션을 구현했습니다.

## 주요 기능

- CSS keyframes를 활용한 프로그레스 바 애니메이션 구현  
- nth-of-type을 활용한 요소별 순차 애니메이션 딜레이 처리  

---

## 기술 스택

- Frontend: HTML5, CSS3

## 담당 역할

## 배운 점 / 인사이트

### CSS 애니메이션 경험

JavaScript 없이도 CSS만으로 단계적인 인터랙션 표현이 가능하다는 것을 경험했습니다.

### 애니메이션 타이밍과 순서 이해

애니메이션에서는 동작 자체보다 타이밍과 순서가 사용자 인식에 큰 영향을 준다는 점을 이해하게 되었습니다.
